Instructions

To use this function, run the following command "./run.sh". To change the maximum value for the sum of prime numbers, you must go into the run.sh file and edit the integer in "java Nathan-Johnson-sumOfPrimes.java ". TEST UPDATE: The tests are a little more complicated. After many errors and exceptions, I was able to get them to work by putting them in the sumOfPrimes class itself (I know it's not the best syntax, but it's what worked). Included is a .jar file for the unit tests. After making test.sh executable (chmod +x test.sh), if you recieve errors regarding the unit test classes not being found, you may need to update the .jar file path in test.sh to the exact path on your cloned copy on your own computer. You may then run ./test.sh INTEGER HERE to run the file.
