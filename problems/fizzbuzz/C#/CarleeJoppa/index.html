Carlee Joppa - Solution - C# - FizzBuzz

How to run FizzBuzz:

  1. Change permissions of run.sh with chmod +x run.sh
  2. run sh run.sh 17 (or some other number, any integer > 0 will work)

How to run tests:

  1. Change permissions of test.sh with `chmod +x test.sh`
  2. run `sh test.sh`h
