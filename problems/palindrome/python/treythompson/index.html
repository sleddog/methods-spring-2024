Instructions

To use this function, you will need to run the command "./runPalindromeCheck.py 'insert your string'"
You may need to run the command "chmod +x ./runPalindromeCheck.py"
This will produce output.

Example

./runPalindromeCheck.py racecar

Output: This is a Palindrome

./runPalindromeCheck.py cat sofa

Output: NOT a Palindrome
