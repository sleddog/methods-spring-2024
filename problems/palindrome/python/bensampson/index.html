Instructions

Run Palindrom Checker

  • First be sure to run the command "chmod +x runTest.sh" to make the program executable.

Example

  • ./runProgram.sh racecar
  • Output: The string racecar is a palindrome!

  • Output: The string camp is not a palindrome!
