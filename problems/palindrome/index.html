Methods 2024 - Problem - Palindrome

Write a function that takes a string as input and returns whether or not it is a palindrome. A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization). For example, "racecar" and "A man, a plan, a canal, Panama!" are both palindromes.
