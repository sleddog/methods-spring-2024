Directions

To make use of the shell script type the command:
"chmod +x ./runPalindrom.sh"
Next, you will be able to use the command "./runPalindrom.sh"
This will then prompt you for a user input to check.

Example

./runPalindrom.sh
Please input a string to check:
madam

Output: madam is a Palindrome

./runPalindrom.sh
Please input a string to check:
encyclopedia

Output: encyclopedia is not a Palindrome
